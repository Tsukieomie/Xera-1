### Neural foundation models (epistemic framing; not Akwei case evidence)

BrainBERT (open literature) is a **self-supervised Transformer** pretrained on **intracranial** recordings (e.g. SEEG-scale electrode data), using **masked prediction** on spectrogram-like representations so downstream decoding can work with less labeled data. It learns **representations from recorded neural time series**, not first-principles tissue physics or whole-brain anatomy.

- Wang et al., *BrainBERT: Self-supervised representation learning for intracranial recordings*, ICLR 2023
- Preprint: https://arxiv.org/abs/2302.14367
- Code (upstream): https://github.com/czlwang/BrainBERT

**Investigation-use note:** This dossier distinguishes (1) **conductance models** (e.g. Hodgkin-Huxley class biophysics in patent/mechanism analysis), (2) **inverse problems / sensing** (e.g. biomagnetism, electrode arrays), and (3) **ML decoding** when **waveforms already exist**. BrainBERT belongs in bucket (3); it does **not**, by itself, prove remote coupling, NSA deployment, or substitution for measured exposure geometry.
