# US3647970A Secrecy-Order Investigation

## Executive summary

The current record supports a cautious finding: US3647970A, “Method and system for simplifying speech waveforms,” was filed by Gillis P. Flanagan on August 29, 1968, issued as U.S. Patent 3,647,970 on March 7, 1972, and concerns conversion of speech waveforms into constant-amplitude square waves for transmission, including secure speech applications ([Google Patents](https://patents.google.com/patent/US3647970A/en)). A non-USPTO-hosted PDF titled “PAT FLANAGAN High Security Speech Scrambler” appears to reproduce a secrecy-order notice and a notice of allowability for the same application number 756,124 and title, including the statement that the application was “in condition for allowance” but would be “withheld from issue” under 35 U.S.C. 181 ([High Security Speech Scrambler PDF](https://img1.wsimg.com/blobby/go/a56ed7e4-a072-4dc8-b939-26ab140309de/downloads/Patrick%20Flanagan%20High%20Security%20Speech%20Scramble.pdf?ver=1685486536436)). ## Secrecy-order framework

Under 35 U.S.C. 181, if publication or grant of a patent would be detrimental to national security, the Commissioner of Patents may order that the invention be kept secret and withhold publication or grant for the period the national interest requires ([U.S. House Office of the Law Revision Counsel](https://uscode.house.gov/view.xhtml?req=granuleid%3AUSC-2007-title35-chapter17&edition=2007)). The same statutory section describes one-year secrecy periods with renewal on agency notification, plus special duration rules during war or a presidentially declared national emergency ([U.S. House Office of the Law Revision Counsel](https://uscode.house.gov/view.xhtml?req=granuleid%3AUSC-2007-title35-chapter17&edition=2007)).

USPTO’s MPEP guidance states that a secrecy order issues when the chief officer of a defense agency notifies the Commissioner for Patents that publication or disclosure by patent grant would be detrimental to national security ([USPTO MPEP 120](https://www.uspto.gov/web/offices/pac/mpep/s120.html)). The same USPTO guidance says an application under secrecy order can proceed in prosecution but, once otherwise ready for allowance, is suspended until the secrecy order is removed, after which the Office may issue a notice of allowance or take other warranted action ([USPTO MPEP 120](https://www.uspto.gov/web/offices/pac/mpep/s120.html)).

## Assessment

The circumstantial timeline is consistent with a secrecy-order delay because the application was filed in August 1968, the scanned notice-of-allowability is reported as mailed January 11, 1971, and the patent issued March 7, 1972 ([Google Patents](https://patents.google.com/patent/US3647970A/en), [High Security Speech Scrambler PDF](https://img1.wsimg.com/blobby/go/a56ed7e4-a072-4dc8-b939-26ab140309de/downloads/Patrick%20Flanagan%20High%20Security%20Speech%20Scramble.pdf?ver=1685486536436)). The legal mechanics also fit: USPTO guidance says an application otherwise in condition for allowance can be suspended until a secrecy order is removed, and the scanned document claims exactly that kind of withholding from issue under 35 U.S.C. 181 ([USPTO MPEP 120](https://www.uspto.gov/web/offices/pac/mpep/s120.html), [High Security Speech Scrambler PDF](https://img1.wsimg.com/blobby/go/a56ed7e4-a072-4dc8-b939-26ab140309de/downloads/Patrick%20Flanagan%20High%20Security%20Speech%20Scramble.pdf?ver=1685486536436)).

The main limitation is provenance: no official USPTO-hosted file-history document confirming the secrecy order was retrieved in this pass, and the direct secrecy-order scan is hosted on a third-party domain rather than a USPTO domain ([High Security Speech Scrambler PDF](https://img1.wsimg.com/blobby/go/a56ed7e4-a072-4dc8-b939-26ab140309de/downloads/Patrick%20Flanagan%20High%20Security%20Speech%20Scramble.pdf?ver=1685486536436)). The USPTO Patent File Wrapper API appears to expose document metadata endpoints for application file histories, but the public API search results indicate API-key requirements, and unauthenticated direct requests returned access-denied responses during this investigation ([USPTO Patent File Wrapper API documentation](https://dlthub.com/context/source/uspto-patent-file-wrapper)).

## Bottom line

The claim that Flanagan’s application 756,124 for “Method and System for Simplifying Speech Waveforms” was subject to a secrecy order is plausible and supported by a direct but third-party-hosted scanned artifact that matches the patent’s application number, title, filing date, and issue data ([High Security Speech Scrambler PDF](https://img1.wsimg.com/blobby/go/a56ed7e4-a072-4dc8-b939-26ab140309de/downloads/Patrick%20Flanagan%20High%20Security%20Speech%20Scramble.pdf?ver=1685486536436), [Google Patents](https://patents.google.com/patent/US3647970A/en)). It is not yet fully confirmed from an official USPTO file wrapper or official archival source in this pass, so the best current wording is: “A third-party-hosted scan appears to show a USPTO secrecy order and withholding-from-issue notice for application 756,124, which later issued as US3647970A; official public patent pages confirm the patent facts but do not independently display the secrecy-order notice.”

## Deeper addendum: why the electronic systems miss this record

Patent Center returning no electronic record should be read as evidence that this is a non-digital paper-wrapper case, not evidence that the wrapper does not exist. The USPTO Open Data Portal Patent File Wrapper dataset is listed with temporal coverage of 2001/2025, which excludes Flanagan’s August 29, 1968 application by decades ([Data.gov ODP PFW metadata](https://catalog.data.gov/dataset/open-data-portal-odp-patent-file-wrapper-pfw-api-search-application-data-continuity-docume)). Patent Client documentation likewise warns that the USPTO ODP covers applications filed on or after January 1, 2001, which makes it structurally unsuitable for application 756,124 ([Patent Client Open Data Portal documentation](https://patent-client.readthedocs.io/en/latest/user_guide/open_data_portal.html)).

The official order route is the USPTO Certified Copy Center. USPTO states that a USPTO.gov account is required to use the Certified Copy Center, and that the relevant order type is “Patent File Wrapper” for a complete case record or “Selected paper from selected file wrapper” for a specific document from the application file ([USPTO Order Certified Copies](https://www.uspto.gov/patents/apply/checking-application-status/order-certified-copies)). The Certified Copy Center FAQ says selected patent-file-wrapper papers require a document type/number, title, date, and description, and that more descriptive information helps the Patent and Trademark Copy Fulfillment Branch locate the correct paper ([USPTO Certified Copy Center FAQ](https://www.uspto.gov/learning-and-resources/frequently-asked-questions-about-certified-copy-center)).

For this investigation, the strongest selected-paper targets are now precise enough to request: (1) “Secrecy Order,” serial/application number 756,124, title “Method and System for Simplifying Speech Waveforms,” applicant Gillis P. Flanagan, alleged secrecy-order date April 25, 1969; and (2) “Notice of Allowability / Form D-10,” mailed January 11, 1971, same application, title, and applicant. USPTO MPEP § 130 says that when a secrecy-order case is in condition for allowance, a notice of allowability Form D-10 is issued and prosecution closes, which matches the structure of the scanned Flanagan notice ([USPTO MPEP 130](https://www.uspto.gov/web/offices/pac/mpep/s130.html)).

The scan’s Form D-10 wording also matches a later public litigation example of USPTO secrecy-order procedure. In *Hornback*, the Federal Circuit described a PTO notice stating that an application was “in condition for allowance” but, “in view of the secrecy order” under 35 U.S.C. 181, would be withheld from issue during the national-interest period; this is useful as a template comparison, but it is not direct evidence about Flanagan or application 756,124 ([Federal Circuit Hornback opinion](https://www.cafc.uscourts.gov/opinions-orders/09-1543.pdf)).

The archival route is NARA Kansas City. NARA states that Record Group 241 includes USPTO patent records and more than 4.5 million patent case files and associated records, and that reproduction requests should provide the patent number, inventor or case parties, patent title, and issue date ([NARA Patent Records](https://www.archives.gov/research/patents)). USPTO separately states that non-digital certified paper file wrappers for patents issued between January 1, 1941 and January 1, 1981 may take up to 47 days, including files located at Suitland, Maryland or NARA Kansas City, which squarely covers US3647970A issued March 7, 1972 ([USPTO paper file wrapper update](https://www.uspto.gov/about-us/news-updates/update-paper-patent-file-wrapper-requests)).

Two apparent digital dead ends are now clearer. USPTO says it has not digitized pre-1995 Official Gazette issues, although an almost complete set of physical copies from January 1872 to September 2002 exists at the Public Search Facility in Alexandria; a 1972 Gazette check may corroborate issuance, but it is unlikely to authenticate the secrecy-order papers themselves ([USPTO Official Gazette](https://www.uspto.gov/learning-and-resources/official-gazette/official-gazette-patents)). Federation of American Scientists secrecy-order statistics are aggregate only and do not provide individual patent or application identifiers, so they cannot authenticate a specific 1969 secrecy order for Flanagan application 756,124 ([FAS Invention Secrecy Activity](https://sgp.fas.org/othergov/invention/stats.html)).

The deepest public-digital route appears exhausted. The next real evidence move is an official USPTO Certified Copy Center or NARA request for either the full paper file wrapper or the two selected papers: the alleged April 25, 1969 secrecy order and the January 11, 1971 Form D-10 notice.

## Flanagan-attributed mentions of the secrecy order

There is at least one first-person-style secrecy-order mention attributed to G. Patrick Flanagan, but it is preserved in a secondary web compilation rather than an official USPTO record. Rex Research reproduces material it identifies as “G. Patrick Flanagan, Ph.D., The Neurophone Mark XI Manual (1979),” including the statement: “Six months after the patent was applied for, the National Security Agency placed the patent application under a top secrecy order #756,124,” followed by: “It took four year, and three law firms to sue for release of my invention. We won the battle, and the secrecy order was rescinded” ([Rex Research](http://www.rexresearch.com/flanagan/neuroph.htm)).

The same Rex Research page also reproduces third-party retellings. In a G. Harry Stine article identified as “Bio Cybernetics Revisited,” *Analog* Magazine, February 1980, Stine states that “at the behest of the Department of Defense, the Department of Commerce placed Flanagan’s new device under secrecy order No. 756,124 dated August 28, 1968,” and that Flanagan was able to get the order rescinded in 1972 so the patent could issue ([Rex Research](http://www.rexresearch.com/flanagan/neuroph.htm)).

The “High Security Speech Scrambler” PDF is different evidence: it does not appear to contain a first-person statement by Flanagan about the secrecy order. It contains an unsigned third-person introduction plus scanned Patent Office-style documents, including a secrecy order for serial number 756,124 and a Form D-10 notice of allowability for the same application ([High Security Speech Scrambler PDF](https://img1.wsimg.com/blobby/go/a56ed7e4-a072-4dc8-b939-26ab140309de/downloads/Patrick%20Flanagan%20High%20Security%20Speech%20Scramble.pdf?ver=1685486536436)).

The principal unresolved issue is chronology. The scanned Form D-10 states “in view of the secrecy order issued April 25, 1969,” while the Stine/Rex Research retelling gives “dated August 28, 1968,” which is the filing-date-adjacent timeframe rather than the Form D-10 secrecy-order date ([High Security Speech Scrambler PDF](https://img1.wsimg.com/blobby/go/a56ed7e4-a072-4dc8-b939-26ab140309de/downloads/Patrick%20Flanagan%20High%20Security%20Speech%20Scramble.pdf?ver=1685486536436), [Rex Research](http://www.rexresearch.com/flanagan/neuroph.htm)). The official file wrapper is still needed to determine whether August 28/29, 1968 reflects filing, referral, classification review, or an error in later retelling, and whether April 25, 1969 is the actual secrecy-order issuance date.

The strongest first-person-style mention of the secrecy order attributed to G. ### 1971 ASPAB Patent Security Category Review List (PSCRL-1) — contextual plausibility, not Flanagan-specific proof

A declassified January 1971 composite patent security category review list, "Patent Security Category Review List" (short title PSCRL-1), prepared by the Armed Services Patent Advisory Board (ASPAB) Sub-Committee with cooperation from the Army, Air Force, Navy, Corps of Engineers, AEC, NASA, and NSA — and assistance from John Utermohle (NSA) — was obtained from the Federation of American Scientists' invention-secrecy mirror ([PSCRL-1 (1971)](https://sgp.fas.org/othergov/invention/pscrl.pdf)). The list was originally classified Confidential and is now Unclassified. It identifies its purpose as the first ASPAB composite category review list of categories of inventions that should be made available in accordance with 35 U.S.C. 181, revising the prior Patent Office composite list ([PSCRL-1 (1971)](https://sgp.fas.org/othergov/invention/pscrl.pdf)).

The relevant Group IX / Item 6A language describes NSA cryptology and communications-security categories including "Voice, both privacy and secrecy, analog or digitalized," communications systems intended to deprive all but the addressee of intelligence, unusual media for transmission, any signals controlled by privacy/secrecy means, exotic modulation means, and digitalized voice and signal parameters ([PSCRL-1 (1971)](https://sgp.fas.org/othergov/invention/pscrl.pdf)). The PSCRL-1 itself does not name individual applications or patents; it is a category list, not a docket.

Interpretation, evidence-graded: PSCRL-1 independently strengthens the plausibility that the subject matter of US3647970A — speech-waveform simplification with explicit secure-speech transmission embodiments — would have fit the contemporaneous review ecosystem reflected in ASPAB/NSA category language. It does **not** prove that application 756,124 was actually referred for review or actually placed under a secrecy order. PSCRL-1 should be cited as agency-context corroboration for the plausibility of the claimed secrecy-order route, and not as independent confirmation of a Flanagan-specific order.

### NARA Schedule N1-241-96-003: Assistant Commissioner for Patents — Group 2200 / Licensing & Review series

The 1996 Assistant Commissioner for Patents records schedule, NARA Schedule N1-241-96-003, refines the named series within Group 2200 / Licensing & Review and is a useful procedural and custody complement to the 1976 schedule ([NARA Schedule N1-241-96-003](https://www.archives.gov/files/records-mgmt/rcs/schedules/departments/department-of-commerce/rg-0241/n1-241-96-003_sf115.pdf)).

- **Item 37 — Patent Applications Selected for Review for Security Issues.** Applications received by Group 2200 for security review by defense agencies and DOE; Group 2200 examiners also determine whether the applications are allowable. Categories include applications under issued secrecy orders ("applications are allowable but patents cannot be issued until secrecy orders are rescinded"), applications abandoned because the subject matter is secret, and cleared applications ([NARA Schedule N1-241-96-003](https://www.archives.gov/files/records-mgmt/rcs/schedules/departments/department-of-commerce/rg-0241/n1-241-96-003_sf115.pdf)).

### Procedural context — FAS "Administration of the Invention Secrecy Act"

The Federation of American Scientists hosts a procedural overview titled "Administration of the Invention Secrecy Act" that aligns with and clarifies the USPTO/NARA records schedules ([FAS — Administration of the Invention Secrecy Act](https://sgp.fas.org/othergov/invention/admin.html)). Extracted procedural points include: the Invention Secrecy Act is administered within the PTO by the Special Laws Administration Group, also referred to as Group 220, with cleared examiners; Group 220 examiners inspect private/no-government-property applications to determine whether they might be detrimental if patented; the Licensing & Review Branch pulls applications and refers them to interested defense agencies; the applications themselves remain physically in the PTO, with microfiche copies sent to defense agencies; each person to whom the application is disclosed signs a dated access acknowledgment that is entered in the application file; and ASPAB served as the conduit for referrals from the PTO to DoD and designated DoD technical experts, with ASPAB recommending a secrecy order if disclosure would be detrimental, generally binding on the PTO ([FAS — Administration of the Invention Secrecy Act](https://sgp.fas.org/othergov/invention/admin.html)). The same source describes the conventional Type 1, Type 2, and Type 3 secrecy-order categories. Type 3 is conceptually the closest fit for a private applicant without a DoD security agreement, but the documentary record does not currently state which type, if any, was applied to Flanagan's application; this note therefore does not assert the order type and treats the typology as procedural context only ([FAS — Administration of the Invention Secrecy Act](https://sgp.fas.org/othergov/invention/admin.html)).
